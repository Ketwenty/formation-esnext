const

const

Le mot clé const permet de créer des constantes.

const PI = 3.14
PI = 12 // Uncaught TypeError: Assignment to constant variable.

Il est possible de modifier les propriétés d’une constante.


const OBJ = {};
OBJ.nom = "NOM";
console.log(OBJ); // Object {nom: "NOM"}
